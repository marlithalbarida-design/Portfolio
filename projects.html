1st Year Projects

Here are some projects I've worked on during my first year as an IT student

Simple Inventory Tracker

A browser-based application to manage and track small business inventory, including stock level, price, and supplier information. It uses local storage to persist data across sessions.

Role: Lead Developer (UI & Logic)
Contributions: Designed the user interface, implemented all CRUD operations, and ensured data validation.
Tools/Languages: HTML, CSS, Vanilla JavaScript, localStorage

Responsive Landing Page

A fully responsive, multi-section landing page for a fictional tech startup, focusing on clean design and mobile-first principles. Features smooth scrolling navigation and modern UI components.

Role: Front-End Designer & Implementer
Contributions: Developed the entire front-end structure, optimized all CSS for desktop, tablet, and mobile views, and implemented the navigation.
Tools/Languages: HTML5, CSS3 (Flexbox/Grid)
